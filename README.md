# FOSS Club 
# Description
This is a simple website that showcases upcoming events organized by the FOSS Club and provides a registration interface for users. It is built using HTML, CSS, and basic JavaScript.

# Features
- Home Page: It Displays club name and navigation menu.
- Events Section: Shows upcoming events with images, titles, dates, and descriptions.
- Register Button: Redirects users to a registration page (`register.html`).
- Contact Section: It Allows users to send a message (form inputs included).
- Responsive Layout: Cards adjust to screen sizes.
